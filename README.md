# Java & Spring 변천사 아카이브

자바(JDK 1.0 ~ Java 25)와 스프링(Spring Framework / Spring Boot)의 역사를 버전별로 정리한 문서 모음이다. 어떤 기능이 언제·왜 추가되었는지, 초기 구조는 어땠는지, 그리고 Kotlin이 Spring에 어떻게 들어왔는지를 다룬다. 모든 문서는 한국어로 작성되었다.

## 구성

- **[java/](java/README.md)** — 자바 변천사. JDK 1.0부터 Java 25까지 버전별 문서(책 형식 인덱스).
- **[spring/](spring/README.md)** — 스프링 변천사. Spring Framework 1.x~6.x, Spring Boot 1.x~3.x, 그리고 Kotlin 통합사.

## 읽는 법

각 폴더의 `README.md`가 그 책의 목차(인덱스)다. 전체 타임라인과 시대 구분을 먼저 보고, 관심 있는 버전 문서로 들어가면 된다.

각 버전 문서는 다음 구조를 따른다:
**릴리스 정보 → 시대적 배경 → 주요 추가 기능(코드 예시 포함) → 그 외 변경 → 영향과 의의 → 참고 출처**

## 버전 관리

각 버전(=파일)을 하나의 커밋으로 관리한다.
